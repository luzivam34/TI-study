Modulo 1 - O que é o ChatGPT

O ChatGPT é um assistente de inteligência artificial baseado em modelos de linguagem (LLMs - Large Language Models).

Ele foi treinado com uma enorme quantidade de textos para aprender padrões da linguagem humana.

Ele consegue:

  • Explicar assuntos
  • Programar
  • Corrigir códigos
  • Escrever textos
  • Resumir livros
  • Criar imagens
  • Analisar documentos
  • Criar tabelas
  • Gerar ideias
  • Traduzir idiomas
  • Ajudar em estudos

Mas ele não "pensa" como uma pessoa. Ele gera respostas com base nos padrões que aprendeu e, quando necessário, pode consultar informações atuais da internet ou usar outras ferramentas disponíveis.

Módulo 2 – Pense nele como um funcionário

Imagine que você contratou um funcionário extremamente inteligente. Se você disser: Faça um sistema. Ele ficará perdido.

Agora observe:

  1. Você é um desenvolvedor Full Stack especialista em Flask.
  2. Quero criar um sistema de estoque.
  3. Banco PostgreSQL.
  4. Backend Flask.
  5. Frontend React.
  6. Explique passo a passo.
  7. Aguarde minha confirmação antes de seguir.

A qualidade da resposta normalmente melhora porque você definiu o contexto e o objetivo.

Módulo 3 – O Prompt

Prompt é tudo aquilo que você escreve para o ChatGPT. Por exemplo:

  • Explique HTML.

Isso é um prompt.

Um prompt melhor seria:

  • Explique HTML como um professor universitário.
  • Utilize exemplos.
  • Faça exercícios.
  • Explique cada tag.
  • Não avance para o próximo assunto sem minha confirmação.

Quanto mais contexto você fornece, mais fácil fica adaptar a resposta ao que você precisa.

Módulo 5 – Conversas possuem contexto

Uma conversa é como uma sala de aula. Você pode dizer:

  • Continuar.

ou

  • Faça igual ao exemplo anterior.

Como o contexto ainda existe, eu consigo acompanhar a conversa.

Mas, se começar uma conversa nova, normalmente é preciso fornecer novamente as informações relevantes.

Módulo 6 – O ChatGPT possui ferramentas

Dependendo da versão e do plano, eu posso utilizar ferramentas além do modelo de linguagem. Por exemplo:

  • pesquisar informações atuais na internet quando isso melhora a resposta;
  • analisar arquivos enviados (PDF, Word, planilhas, imagens);
  • ajudar com código e, em alguns casos, executá-lo para gerar tabelas, gráficos ou documentos;
  • gerar e editar imagens;
  • criar lembretes e automações (quando disponíveis na sua conta).

Nem toda conversa exige essas ferramentas; elas são usadas quando fazem sentido para a sua solicitação.

Módulo 7 – Memória

Existe uma diferença importante entre:

Memória

Informações que você pediu para eu guardar para conversas futuras (como preferências de estudo, estilo de resposta etc.).

Exemplo:

Sempre explique programação passo a passo.

Se você pedir para eu lembrar disso, posso usar essa preferência em conversas futuras.

Contexto

É tudo aquilo que está sendo discutido na conversa atual. Quando a conversa termina, esse contexto não é automaticamente mantido. Apenas informações que você pediu

para memorizar (ou que foram salvas conforme as configurações da sua conta) podem permanecer.

Módulo 9 – O segredo dos usuários avançados Em vez de fazer uma única pergunta grande, eles costumam trabalhar em etapas. Por exemplo, em vez de pedir "crie uma Pokédex completa", eles fazem: Planejamento. Estrutura das pastas. Banco de dados. Backend. API. Frontend. CSS. Testes. Deploy.
